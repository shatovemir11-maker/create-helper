# Create Helper

Мобильное Android-приложение для обучения механике мода **Create** в игре Minecraft.

## Функции
- 7 пошаговых уроков по технической механике
- Прогресс-бар и сохранение прогресса
- Отметка пройденных уроков (зелёная галочка)
- Работает на Android 7.0+ (API 24)

## Технологии
- Kotlin
- Android SDK (View-система)
- Room (локальная база данных)
- Coroutines (асинхронность)
- RecyclerView (списки)
- KSP (кодогенерация)
